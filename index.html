Bootcamp 2020: Hangman game

This is my first Processing project summarizing the 2 week Programming Bootcamp at City University (2020). Program, to run properly uses multiple Processing tools like classes, loops, conditions, methods and randomness. This game has the option to be played by one person or two - this can be chosen after the start of the program by chosing a correct button. Chosing a singleplayer mode will randomly chose the guessing word from the provided text file and start a game. Multiplayer mode will first allow you to input the word that later will be guessed by another player. It is submitted by clicking enter.
Whent the game is played, each key press starts the sequence of methods that determin wheter the inputed letter is right or wrong, and acts accordingly. If the letter was correct, program displays it. If not, than the part of the gibbet is drawn.
The game ends when the wole word is guessed or the whole gibbet is displayed on the screen. Then the player has a choice to play again with a different word of to exit.
